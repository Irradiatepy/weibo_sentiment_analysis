# weibo sentiment analysis
Hi everybody，

这是我毕设项目的源码，题目是《基于社交网络的情绪化分析》。意义是：使用数据分析的方法，从数学的角度去研究在社交网络上人们表达情绪的倾向。

--------

## 文件解释

- weibo_download:
	给出了下载微博数据的全部代码，配置文件，数据库表结构。

- weibo_analysis:
	给出了分析微博情感倾向的全部代码。分析过程中涉及到大量的数据（我将很多数据都转存成pickle文件），这些数据在上传代码时被我删掉了。
